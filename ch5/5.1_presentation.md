## question 1

        A = {5,7,8,9,10} and B = {5,10,11,15,20} Find each of the following sets.

a. $$A \cup B$$

Add both of all the element to or each set
= {5,7,8,9,10,11,15,20}


b. $$A \cap B$$

Only the element that are in each set
= {5,10}

c. A \ B

All A that are not in B
 = {7,8,9}

d. B \ A

All B that are not A
 = {11,15,20}
